# Movie Recommendation System using sentiment Analysis & Collaboration Filtering 

## Project Description
This project recommends movies based on user sentiment analysis from tweets and user ratings.

## Technologies Used
- Python
- Machine Learning
- Sentiment Analysis
- NLP
- Pandas
- Numpy
- Content-Based Filtering
- Collaborative Filtering

## Project Structure
- dataset/
  - movies.csv
  - ratings.csv
  - tweets.csv
  - users.csv
- code/
  - MovieRecommendation.py
  - TweetReader.py
  - run.bat
- README.md

## How It Works
1. Reads user tweets and reviews
2. Performs sentiment analysis
3. Recommends movies based on positive sentiment and ratings

## How to Run
1. Install Python
2. Install required libraries
3. Navigate to code and Run MovieRecommendation.py or run.bat

## Output
Generates personalized movie recommendations.
